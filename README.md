# Non-Linear & Advanced Data Structures

A complete collection of **Non-Linear and Advanced Data Structures** focused on trees, graphs, hashing, priority queues, tries, and probabilistic data structures.
This repository provides practical implementations, traversal techniques, collision handling, recursion concepts, and efficient searching methods used in modern computer science and real-world systems.

---

# 📚 Topics Covered

---

## 20. Binary Search Tree (BST): Insert / Search / Inorder Traversal

Implementation of **Binary Search Tree (BST)** with insertion, searching, and inorder traversal.

### Features

* Insert nodes
* Search elements
* Inorder traversal
* Sorted output proof
* Recursive implementation

### Concepts Learned

* Binary tree structure
* Ordered data storage
* Recursive traversal
* Efficient searching

### Important Note

Inorder traversal of BST always produces elements in sorted order.

### Complexity Highlights

* Average Search → O(log n)
* Worst Case → O(n)
